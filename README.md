### Creating another one. Requirements:   
Check php version   
```php --version```   
Install curl   
```sudo apt-get install php7.4-curl```   
Add domain in etc/hosts:   
```sudo nano /etc/hosts```   
127.0.0.1 localhost yourdomain.com   
```sudo apt install php7.4-mbstring   
sudo systemctl restart apache2```
